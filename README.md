# 🏥 Health Care Chatbot (AI आधारित Disease Prediction System)

## 🚀 Overview
The **Health Care Chatbot** is a Machine Learning-based application that predicts possible diseases based on user-input symptoms. It also provides detailed disease descriptions, precautionary measures, and voice-based responses for better user interaction.

This project demonstrates how AI can be applied in the healthcare domain to assist users with basic medical insights.

---

## 🎯 Features
- 🤖 Disease prediction using Machine Learning (Decision Tree Classifier)
- 📊 Trained on real-world symptom-disease dataset
- 🖥️ User-friendly GUI built with Tkinter
- 🔊 Text-to-Speech integration using pyttsx3
- 📋 Displays:
  - Predicted Disease
  - Description
  - Precautions

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Libraries:** pandas, numpy, scikit-learn, pyttsx3  
- **GUI:** Tkinter  
- **ML Model:** Decision Tree Classifier

📊 How It Works
Dataset is used to train a Decision Tree Classifier
Symptoms are encoded into numerical values
Model predicts the most probable disease
Information is fetched from master datasets
Results are displayed via GUI and voice output
